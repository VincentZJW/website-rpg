# BGM Asset Notes

This directory contains generated original background music and may also contain legally usable replacement files.

Recommended formats:

- `.ogg`
- `.mp3`
- `.webm`

Run `npm run generate:rpg-bgm` to recreate the six original soft RPG-town WAV files:

- `/audio/bgm/adventurers-guild-morning.wav`
- `/audio/bgm/willow-town-market.wav`
- `/audio/bgm/moonlit-library-waltz.wav`
- `/audio/bgm/robotics-scout-path.wav`
- `/audio/bgm/career-fog-guardian.wav`
- `/audio/bgm/spawn-obelisk-awakening.wav`

The dependency-free Node.js generator uses soft synthesized flute-like leads, strings, harp or pluck notes, bells, pads, gentle bass, light percussion, and short echoes. The tracks are programmatically generated and project-original: no OpenGameArt file or official commercial game music is downloaded or reproduced. Missing files are safe: the site automatically uses original WebAudio loops when a configured file is absent or cannot be played.

See `docs/rpg-bgm-tool-flow.md` for the reusable generation workflow.

## Optional Licensed Replacement File

`/audio/bgm/user-provided-littleroot-town.mp3` is reserved for a file you provide yourself. No real file is included or generated by this project.

Only add and enable a replacement recording if you have permission to use it. Do not commit unauthorized commercial game music to a public repository.

After adding a legally usable file, update `townTrackOverride` in `lib/bgm-config.ts`:

```ts
townTrackOverride: "licensedLittlerootTown",
```

Set it back to `null` to disable the override.

## Royalty-Free Alternatives

Download assets manually and verify the license for each individual track before use:

- OpenGameArt: search for CC0 town theme RPG music
- OpenGameArt: search for CC0 fantasy music and sounds collections
- Pixabay: search for fantasy, RPG, or fantasy city royalty-free music
- itch.io: search for free fantasy RPG game tracks and check the author's license

After downloading a track, place it in this directory and update the corresponding `src` in `lib/bgm-config.ts`.

Do not download or bundle unauthorized Pokémon, Nintendo, Game Freak, or The Pokémon Company music files.
